SUPINFO AppFactory Plugins SDK

Here you will find the help you need to start building your plugin.

    Submit your plugin

  • Once your plugin is ready to be integrated to your app. You only need to zip the folder and send it to us to the following address: appfactoryplugins@supinfo.com
  • You will receive a License key with which you can install your plugin and use it inside your app.
  • Make sure you run your local environment as a web application environment and not as static files from the file system.
